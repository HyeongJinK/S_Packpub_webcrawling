Getting started

Let's start by installing Mesosphere and its components on our AWS server. Firstly, log in to your AWS server.

You should see your internal IP address. To log in, you can use a public IP, and then, when an instance is generated, a public IP will be assigned automatically. We can now move ahead to installing the master server, but first, we need to change the hostname. The next section will tell you how this can be done.

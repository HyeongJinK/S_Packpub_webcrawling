Chapter 9. Classification of Remote Sensing Images

This chapter will familiarize you with the concept of raster image classification using the Semi-Automatic Classification Plugin (SCP) of QGIS.There are two major types of classification: supervised and unsupervised. We will cover supervised classification here, which uses training data to classify the whole dataset or a raster image. Learning about this will help to classify land cover according to different requirements or characteristics.

The following topics will be covered in this chapter:

  • Definition of supervised classification
  • Supervised classification in QGIS
  • Creating validation data in QGIS

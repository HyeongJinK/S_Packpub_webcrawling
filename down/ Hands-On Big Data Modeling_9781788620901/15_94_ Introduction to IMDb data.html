Introduction to IMDb data

IMDb provides a subset of datasets for use by personal and enterprise applications. Subject to their terms and conditions, IMDb can be used for personal and non-commercial purposes. It is available for download at https://datasets.imdbws.com/. 

In this mini project, we are using two types of subset:

  • Episode data: Contains the TV episode information
  • Rating data: Contains IMDb ratings and vote information for titles

 

Episode data

Episode data contains TV episode information. It contains the following fields:

tconst

 A string field that contains an alphanumeric identifier of the episode. 

parentTconst

 A string field that contains an alphanumeric identifier of the parent TV series. 

seasonNumber

 An integer field. It is the season number the episode belongs to. 

episodeNumber

 An integer field. It contains the episode number of tconst in the TV series.

 

This dataset was downloaded from https://datasets.imdbws.com/title.episode.tsv.gz.

Rating data

This contains the IMDb ratings and the vote information for titles. It contains the following fields:

tconst

 A string field. It is an alphanumeric unique identifier for the title.

averageRating

 The weighted average of all the individual user ratings.

numVotes

 The number of votes the title has received.

 

The purpose of this exercise is to find important factors affecting the average ratings of the entries in the IMDb dataset. We will use clustering to find the most important factors. We guess that a larger number of episodes and votes can lead to a higher average rating.

This dataset was downloaded from https://datasets.imdbws.com/title.ratings.tsv.gz.

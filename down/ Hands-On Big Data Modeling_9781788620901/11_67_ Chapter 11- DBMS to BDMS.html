Chapter 11. DBMS to BDMS

As you know, managing big data requires a different approach to database-management systems because of the wide variation in data structure and data sources, which does not lend itself well to traditional DBMS. There are many applications available to help with big data management. This lesson introduces you to some of these applications and provides insight into how and when they might be appropriate for the big data management challenges.

The main sections discussed in this chapter include:

  • Explaining characteristics of Big Data Management System (BDMS)
  • Exploring data management with Remote Dictionary Server (Redis)
  • Using Aerospike as a new-generation key-value store
  • Using AsterixDB for studying semi-structured data

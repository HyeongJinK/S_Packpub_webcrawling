Chapter 6. Deploying Solutions to Azure

Now that we have gained an understanding of developing solutions for Azure, how do we deploy these solutions? In this chapter, we will learn what you need to know about deploying solutions in Azure. We will approach this using DevOps, which can sit on top of any Agile process you wish to use for your development processes.

The following topics will be covered in this chapter the following:

  • Deploying to Azure
  • Testing deploys
  • Using deployment slots
  • Deploying infrastructure

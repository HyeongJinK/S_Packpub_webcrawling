More tween components

We have added and configured a variety of tween components. Here's a list of all other available tween components that work in a similar way:

  • Tween Color

  • Tween Rotation

  • Tween Transform

  • Tween Height

  • Tween Width

  • Tween Volume

  • Tween Field Of View

  • Tween Orthographic Size

These tween components can be added to GameObjects and are configured in the same way as the ones we have already used in this chapter. They will come in handy later on.

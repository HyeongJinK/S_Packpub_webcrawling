Chapter 2. Using Textures for Effects

In this chapter we start looking at ways in which you can use textures to create different effects in your Shaders. As we saw in the previous chapter, textures can help us achieve more complex lighting effects. We can also use texture to animate, to blend and really, to drive any other property we want. In this chapter we learn about the following methods:

  • Scrolling textures by modifying UV values

  • Animating sprite sheets

  • Packing and blending textures

  • Normal mapping

  • Creating procedural textures in the Unity editor

  • Photoshop levels effect

Chapter 3. Making Your Game Shine with Specular

Everybody loves games like Gears of War and Call of Duty, but what is it about these games that make them so visually compelling and very realistic? Well, it is a combination of things really, but one of the more key elements that these games employ in their Shader pipelines are different types of Specular. This chapter will introduce you to the basics of Specular and demonstrate some of the tricks that today's AAA games use every day in their Shader pipelines.

In this chapter, will learn the following:

  • Utilizing Unity3D's built-in Specular type

  • Creating a Phong Specular type

  • Creating a BlinnPhong Specular type

  • Masking Specular with textures

  • Metallic versus soft Specular

  • Creating an Anisotropic Specular type

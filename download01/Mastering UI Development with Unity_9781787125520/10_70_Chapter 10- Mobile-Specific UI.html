Chapter 10. Mobile-Specific UI

If youare creating a game for multiple platforms, you really have to consider making interfaces that either work on all platforms you plan on developing for, or make device-specific interfaces. One advantage of the Unity Event System is that it translates very well to mobile devices. An action such as clicking a button with a mouse translates perfectly to a touch or tap gesture. There are a few interactions that don't translate, however, such as keyboard key presses, right-clicking, and scrolling the mouse wheel. Let's discuss a few items of UI design that are specific to mobile/touchscreen devices.

In this chapter, we will discuss the following topics:

  • The recommended button sizes for mobile games
  • Laying out interactions based on the thumb zone
  • How to access multi-touch
  • When to use the accelerometer and gyroscope
  • Creating on-screen D-pads

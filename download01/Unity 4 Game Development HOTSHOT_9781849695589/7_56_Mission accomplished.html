Mission accomplished

In this project, we first created the ragdoll object and applied this to our AI character. Then, we created the destructible Wall object and destroyed it when we shot at it by adding some script to the rocket script. We also created the Rockslide game object, the Rocks script to enable and disable Rigidbody of the rocks, and the TriggerArea game object and RocksTrigger script to make the rocks fall down when the player hits the trigger area.

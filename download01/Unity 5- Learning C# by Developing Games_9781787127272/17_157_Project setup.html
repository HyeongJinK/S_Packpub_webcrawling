Project setup

At this point, I have assumed that you have started Unity up:

  1. With Unity started, go to File | New Project. Select a Project Location of your choice somewhere on your hard drive, ensure that you have 2D selected, and under Project Name, change it to Tappy Plane. Once completed, select Create Project. From there if you see the Welcome to Unity popup, feel free to close it as we won't be using it.

  2. Create the following folders just as we described in the previous chapters:

    • Prefabs

    • Scenes

    • Scripts

    • Sprites

Summary

In this chapter, we went through the background of game development and were introduced to the game engine we will be using, namely Unity Engine.

We also learned how to work with Unity's new 2D workflow by creating a 2D scene using the new project option available and then stepped through the six main Editor Views that make up the Unity Editor interface. We also went through the additional Views that are available.

In the next chapter, we will look at setting up our own scene. We will set up all the elements that make up our environment, namely the camera and the background.

Survival

In an endless runner, the character usually only has one life, and as such, we need to check our game. Check when the ninja collides with an obstacle, because when that happens, he dies and the game ends.

Note

A Unity function to detect whether the ninja has collided with an obstacle is declared next. We do not need to call the function since it is part of the Unity API and gets called automatically by Unity's Physics engine.

function OnCollisionEnter2D(other:Collision2D)
{

Note

If the name of the object that has collided with the obstacle is Ninja, then end the game by stopping the game using Time.timeScale = 0;. And then, destroy the obstacle. Have a look at the next code snippet.

    if(other.gameObject.name=="Ninja")
    {
        Time.timeScale = 0;
    Destroy(gameObject); 
  }
}

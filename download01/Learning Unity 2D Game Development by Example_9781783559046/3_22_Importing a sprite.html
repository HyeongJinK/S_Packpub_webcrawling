Importing a sprite

To import a sprite, we repeat the process that we followed when importing a background. We drag the sprite from its location on our PC into the Project Browser and then set its Texture Type option to Sprite within the Inspector view. The only difference is that this time, we set the Sprite Mode drop-down box to Multiple since our single character sprite is made up of multiple images. Once we are done setting the import settings, we hit Apply.

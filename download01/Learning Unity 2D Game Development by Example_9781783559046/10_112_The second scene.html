The second scene

To learn and practice using multiple levels, we will add a second scene. For the second scene, let's execute the following steps:

  1. Create a new scene and save it as Scene2.

  2. We move this scene into the Scenes folder in the Project Browser.

  3. We repeat all the preceding steps except that we use the alternative second scene sprites.

  4. We name all the objects the exact same names as in the first scene except for the UFObeam GameObject, which we name UFObeam2 because it has unique code associated with it.

Detecting collisions

In order to detect any collisions, we need to ensure that our hero, orb, and enemy GameObjects have Box Collider 2D components added to them.

When the magical orbs that we just added collide with an enemy, we want to destroy the enemy by detecting if the colliders of both the enemy and the orb collide.

Add the following code to the enemy script:

function OnCollisionEnter2D(other:Collision2D)
{
  if(other.gameObject.name=="orb(Clone)")
  {
    Destroy(other.gameObject);
    Destroy(gameObject);


  }

}

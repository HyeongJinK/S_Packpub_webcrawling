I want to create a 3D game

Optimization of 3D games is often much more intensive than a 2D game because you will be handling scenes that are much larger and more detailed. When you create 3D models for an iOS game, it is important to follow a couple of rules.

Make the mesh as low detailed as possible, although it should still be pleasant to look at. The more triangles the mesh has, the more difficult it is for the device to render it.

Use as few materials as possible. Similar to all devices, each object is rendered to the screen on each frame. The more objects you have, the more time is spent rendering them. For iOS games, the objects being rendered are called draw calls. The fewer draw calls you have, the better the game will run. If you have too many draw calls, the devices will use too much memory and/or take too long to render, which will either outright crash the game with an out of memory message or start to render at lower frames-per-second, making the game appear slower than it should be.

3D programs

Many mobile games are in 3D and require the use a 3D modeling program. A common free option is called Blender.

You can find more information about Blender at https://www.blender.org/.

Another option is from Autodesk. This has both Maya and 3D Studio Max. Both cost money, have month-to-month licensing plans, and allows you to pay for a commercially acceptable license over a smaller period of time. Refer to http://www.autodesk.com/products/maya/overview for more information on Maya.

You can find more information about 3Ds Max at http://www.autodesk.com/products/3ds-max/overview.

All of these 3D programs also offer animation tools, such as creating skeletons and handling of animation controls. Animation is well documented online. Also, there are many video tutorial series for each of these programs on how to animate a static 3D model.

These programs also provide you tools for UV mapping, which is the process of taking faces of the model and laying them out in a flat sheet to make textures fit the mesh correctly. UV mapping is well documented online. Also, a Google search will give you literally thousands of resources on how to do it.

Creating textures

A commonly used option is a program called Photoshop created by Adobe. Adobe provides a license for Photoshop that is very cheap; currently, it is $9.99 a month for Photoshop and Lightroom.

You can find more information about Photoshop at https://creative.adobe.com/plans?promoid=KSPAK.

Photoshop tutorials have a massive presence online. You can also find many excellent resources through a Google search.

Two more resources on how to make textures a bit easier are DDO/NDO (created by Quixel) and Substance Painter / Substance Designer (created by Allegorithmic).

You can find more information about DDO/NDO at http://quixel.se/dev/ddo.

Refer to https://www.allegorithmic.com/products/substance-live for more information on substance tools.

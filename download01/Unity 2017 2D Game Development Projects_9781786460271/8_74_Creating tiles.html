Creating tiles

One of the most iconic things about 2D games is the use of tiles. Just like tiles within your house, combining tiles together can create patterns, and those patterns, within a game's context, can result in its levels. In this chapter, we will learn just how we can use tiles in a third-party program called Tiled, and how to then take the levels that we build and import them into Unity using the program Tiled2Unity. These are discussed more in just a moment.

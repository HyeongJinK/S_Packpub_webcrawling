Chapter 11. Don’t Forget to Save!

In this chapter, we continue building the game by giving suggestions about how to script gameplay. In addition, you will learn how to create and implement a basic save and load system into your game. Before concluding the chapter, we will extend the Movement Component to include Jump pads, to add a bit more pizzazz to your game. Lastly, this chapter covers some general topics for testing platformer games, which includes what to look for to ensure that everything is running correctly.

In particular, the chapter will cover:

  • Save and load systems
  • Jump pads
  • Suggestions on how to finish up the gameplay of the game

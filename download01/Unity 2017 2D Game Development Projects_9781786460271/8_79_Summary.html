Summary

In this chapter, we have covered how to create tiles using the third-party program Tiled. We have learned how to set up different layers (such as Terrain and Water), so to better distinguish the different parts of the map once we import everything back into Unity. We used another tool named Tiled2Unity to export the map created in Tiled into Unity and learned how to set the environment up. We will continue with the environment in Chapter 10, Let's Get Physical. But for now, it's time to jump into Chapter 9, Look, It Moves, in which we will learn more about the Animation State machine, and we will build a more complex one.

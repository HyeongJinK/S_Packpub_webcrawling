Setting up the sprite sheet for the animations

If you recall what we did for the explosion, we had an animation sprite sheet with a single animation. Here instead, if you take panda_animation_spritesheet.png within our asset package, we have more than one animation:

The image above is the Sprite Sheet that we will be using for our Panda animations. As you can see, there are many different types of animations for various situations.

So, let's go to the panda_animation_spritesheet Import Settings, and you should see something like the following:

However, we need to set the Texture Type to Sprite (2D and UI), as well as set the Sprite Mode to Multiple, as shown here:

Then, open the Sprite Editor. In the Slice menu, select Type as Grid by Cell Count (since the sprite sheet has every animation frame in a specific cell). In particular, slice it into 6 columns by 17 rows. An example of this is shown in the following screenshot:

The following screenshot shows the end result. Here, you will faintly see a bounding box around each frame of the panda's animation:

An example of the Panda animation Sprite Sheet after it has been sliced.

Now we have all the animations frames done, it's time to transform them into single animations.

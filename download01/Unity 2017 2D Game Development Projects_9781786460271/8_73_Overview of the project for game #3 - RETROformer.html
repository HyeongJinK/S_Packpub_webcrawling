Overview of the project for game #3 - RETROformer

Like the other projects, we will go through how to set up RETROformer.

We will be using two programs: Tiled and Tiled2Unity in conjunction with Unity to create RETROformer. Here is a brief overview of the roles that each will play in the development of RETROformer:

  • Tiled: Where we will create the level environment that the player will navigate through
  • Tiled2Unity: Where we will import the level created in Tiled into Unity
  • Unity: Where we will implement player controls, a save system, various other interactive, decorative (for example, trees, background effects, and so on), and UI elements

At this stage you should be used to the file structures for creating games from the last 2. So for the third game, you can use the same folder structure that you have for the previous games.
